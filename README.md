Infant Nutrition Support System

Overview:
The Infant Nutrition Support System is a Java-based desktop application using Java Swing for the frontend and MySQL for the backend. It connects low-income families with NGOs and donors to provide infant nutrition kits. The system tracks beneficiaries, donors, nutrition kits, and distribution, ensuring an organized and transparent process.

Technologies Used:
Java (Backend)
Java Swing (GUI Frontend)
MySQL (Database)
JDBC (Database Connection)
Eclipse IDE

Modules:
1.Admin Module
Add or manage NGOs and donors
View registered families
Allocate nutrition kits
Generate reports

2.Beneficiary Registration
Register low-income families
Collect details including mother’s name, baby age, income, address, and phone

3.Donor / NGO Module
Register donors and NGOs
Track donated kits

4.Nutrition Kit Tracking
Store kit details including name, quantity, and expiry date
Track available stock

5.Distribution Tracking
Record which family received kits
Track who distributed the kits and when
